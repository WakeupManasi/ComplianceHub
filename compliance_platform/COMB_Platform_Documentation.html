⚙ Platform Overview

COMB is an AI-powered Django platform that automates monitoring and analysis of regulatory changes in the Indian financial and industrial landscape. It uses 7 autonomous AI agents working in a pipeline to reduce the time from "Regulation Published" to "Impact Report Generated" from days to minutes.

Regulatory Intelligence

Automated scraping of RBI, SEBI circulars. Clause-level diff analysis and impact mapping across 8 compliance categories.

Threat Intelligence

Dark web monitoring, leaked credential detection, domain scanning with industry-specific profiles across 17 sectors.

CVE Management

60+ realistic CVEs (2023-2026) with CVSS scores, affected products, and severity tracking.

Multi-Industry Support

18 organizations across Banking, Healthcare, Defence, Manufacturing, Pharma, Energy, NBFC, Legal, and more.

🔍 Enhanced Domain Scanning

Domain scanning now includes 17 industry-specific profiles with tailored email patterns, subdomain discovery, tech stack fingerprinting, and DNS/SSL analysis.

Scan Features

Industry-Aware Email Scanning

Each industry has custom email prefixes (e.g., treasury@, scada@, clinical-trials@) in addition to standard patterns.

Subdomain Discovery

Simulates subdomain enumeration with industry-specific targets (e.g., netbanking, scada, ehr, claims-portal).

Tech Stack Fingerprinting

Detects industry-relevant technologies (e.g., SWIFT Alliance for banking, SCADA/HMI for manufacturing).

DNS/SSL Analysis

Checks SPF records, DMARC policies, dangling CNAMEs, and SSL certificate expiry.

Extended Breach Sources

12 base sources + industry-specific ones (e.g., "Financial Sector Stealer Logs", "ICS-CERT Advisory").

Risk-Weighted Scoring

Industry-specific breach probability (30%-50%) and enhanced risk calculation including subdomain and DNS findings.

💻 Setup & Running

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py makemigrations
python manage.py migrate

# Seed all test data (accounts, CVEs, dark web intel)
python manage.py seed_threat_data

# Run the server
python manage.py runserver
